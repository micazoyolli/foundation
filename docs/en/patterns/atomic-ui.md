# Atomic UI — Future

Foundation Core does not include visual components. This page defines future criteria without creating fictional APIs.

## Current state

- No React components.
- No buttons.
- No cards.
- No layouts.
- No brand tokens.
- No final typography.

## Atoms

Reserved naming examples: `Button`, `IconButton`, `TextInput`, `VisuallyHidden`. They are not exports today.

## Molecules

Reserved naming examples: `SearchField`, `MediaGuard`, `MetadataPreview`. They are not exports today.

## Organisms

Reserved naming examples: `Header`, `Sidebar`, `ProductGrid`. They are not exports today.

## Requirements

Repeated production use, validated accessibility, small stable props, no brand tokens, documentation and tests/smoke when relevant.
